# 源氏连招进阶指南 — 从青铜到宗师

> 源氏（Genji）是守望先锋中最具操作上限的英雄之一。本文将从基础连招到高阶技巧，帮你全面提升。

---

## 一、基础连招

### 1.1 标准爆发
```
右键（三镖）→ 闪（Shift）→ 近战（V）
```
**伤害**：84 + 50 + 30 = **164**，足够秒杀 200 血脆皮。

### 1.2 远程消耗
- 左键精准点射
- 利用镖速预判走位
- 中距离 **右键散射** 压血线

---

## 二、高阶技巧

| 技巧 | 难度 | 说明 |
|------|------|------|
| 影闪取消 | ⭐⭐⭐ | `Shift` 击杀重置后立即再 `Shift` |
| 爬墙镖 | ⭐⭐⭐⭐ | 爬墙时出镖，利用高度差命中头部 |
| 拔刀跳劈 | ⭐⭐⭐⭐⭐ | 开大时跳 → `Shift` → 空中砍 → 落地砍 |

### 💡 核心原则
1. **不要第一个冲** — 等队友开团
2. **留着 `Shift`** — 除非能击杀重置
3. **爬墙 = 第二条命** — 二段跳 + 爬墙无限位移
